PacMen Factory

Working with array methods, callback functions, and DOM manipulation was the focus of this project. I created a web page that allows the user to randomly generate PacMen at the click of a button and initiate the movement of each PacMan with the click of another button. This was done by first creating an empty array. Once the user clicks on the "generate" button, a for loop is executed that pushes a new PacMan object into the empty array for each iteration.

Real Time Bus Tracking

This project uses asynchronous programming, DOM manipulation, and the use of recursive functions. Asynchronous programming is a common technique used when fetching data from an API. The async/await function allows the program to continue running while the data is being fetched, and then manipulate the data once it has been received. This Project also involved using recursive functions in order to ensure that the map markers continued to update even if there was an error with fetching the data.

WHY MERN?

The MERN stack is a comprehensive toolkit for building modern web applications. It includes everything you need to get started, from the MongoDB database to the React front-end framework. With its high-performance technologies and customizable components, the MERN stack provides all the tools you need to build scalable, feature-rich web applications.
