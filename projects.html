RECENT PROGRAMMING EXERCISES



PacMen Factory

Working with array methods, callback functions, and DOM manipulation was the focus of this project. I created a web page that allows the user to randomly generate PacMen at the click of a button and initiate the movement of each PacMan with the click of another button. This was done by first creating an empty array. Once the user clicks on the "generate" button, a for loop is executed that pushes a new PacMan object into the empty array for each iteration.
